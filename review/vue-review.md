# vue 的内容复习

## 提问：简单介绍一下vue框架
回答：vue是基于mvvm模型的渐进式组件化前端框架，特点有双向数据绑定，指令使用，组件的生命周期，

## 提问：什么是mvvm模型
回答：mvvm是由model、view、viewModel组成，视图和数据都与viewModel关联，一方改变通过vm触发另一个改变
